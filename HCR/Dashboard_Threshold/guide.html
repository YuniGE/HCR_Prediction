ESI Threshold Dashboard 使用指南

📌 关键概念解释

两种"年份"的区别（非常重要！）

概念 英文 范围 含义
发布日期 Released Date 2017-2025 ESI 发布阈值数据的日期（约每2个月更新一次）
出版年份 Publication Year 2007-2025 论文原始发表的年份

关键理解： - 你的数据包含 49 次 ESI 发布（从 2017年10月 到 2025年11月） - 每次发布包含约 10-11 个出版年份 的阈值 - 例如：2024年11月的发布 包含 2015-2025 年出版的论文的阈值

为什么 Multi-Year Comparison 显示 2016-2024？

这显示的是 出版年份（Publication Year），不是发布日期： - 2016年出版的论文 → 在你的数据期间（2017-2025发布）一直被跟踪 - 2024年出版的论文 → 只有最近几次发布才有数据

图表展示的是： 追踪某一年出版的论文，随着时间推移，其阈值如何增长。

📊 各图表详细解释

1. Time Series Analysis（时间序列分析）

显示内容： 选定出版年份的论文，其阈值随 ESI 发布时间的变化

横轴： ESI 发布日期（Release Date）
纵轴： 阈值（Threshold）

解读示例： - 选择 Publication Year = 2020 - 图表显示 2020年出版的论文，从第一次被 ESI 收录到最新发布，阈值如何从 3 增长到 168 - 曲线上升说明：随着时间推移，2020年的论文积累了更多引用

实际意义： 如果你2020年发表的论文有100次引用，查看这条曲线可以知道在哪个时间点你的论文超过了阈值。

2. Multi-Year Comparison（多年对比）

显示内容： 比较不同出版年份（2016, 2018, 2020, 2022）的阈值演变

为什么选这些年份： - 展示不同"年龄"论文的阈值差异 - 2016年出版的论文比2022年出版的论文"老"6年 - 老论文有更多时间积累引用 → 阈值更高

关键洞察： - 所有曲线都在上升（随时间积累引用） - 老的出版年份（2016）曲线在上方 - 新的出版年份（2022）曲线在下方

3. Age-Threshold Relationship（年龄-阈值关系）

显示内容： 论文年龄与阈值的平均关系

论文年龄计算： Paper Age = Release Year - Publication Year

示例： - 2020年出版的论文，在2024年ESI发布时，年龄 = 2024 - 2020 = 4年 - 2015年出版的论文，在2024年ESI发布时，年龄 = 2024 - 2015 = 9年

关键发现： 阈值遵循 幂增长模式：

Threshold ≈ a × Age^b + c
  • 年龄越大，阈值越高，但增长速度逐渐放缓

4. Growth Patterns by Field（领域增长模式）

显示内容： 不同研究领域的阈值增长曲线

关键差异： - Clinical Medicine: 高引用领域，曲线陡峭 - Mathematics: 低引用领域，曲线平缓

实际意义： 不同领域"高被引"的标准差异巨大 - 在临床医学中需要200+引用才是高被引 - 在数学中可能50次引用就是高被引

5. Average Threshold by Field（领域平均阈值）

显示内容： 各领域的整体平均阈值

排序： 从低到高

高引用领域（阈值高）： - Molecular Biology & Genetics - Clinical Medicine - Multidisciplinary

低引用领域（阈值低）： - Mathematics - Computer Science - Economics & Business

6. Current Thresholds（当前阈值）

显示内容： 最新一期 ESI 发布中，选定出版年份在各领域的阈值

使用场景： - 想知道你2020年发表在化学领域的论文现在需要多少引用才是高被引？ - 选择 Publication Year = 2020，查看 Chemistry 的条形高度

7. Threshold Heatmap（阈值热力图）

显示内容： 完整的 领域 × 出版年份 矩阵

颜色： - 浅色 = 低阈值 - 深蓝色 = 高阈值

模式解读： - 左边列（老出版年份）颜色深 → 老论文阈值高 - 右边列（新出版年份）颜色浅 → 新论文阈值低 - 上方行（高引用领域）颜色深 - 下方行（低引用领域）颜色浅

8. Predicted vs Current（预测 vs 当前）

显示内容： - 柱状图：2026年1月预测阈值 - 折线图：最新实际阈值

预测模型： Gradient Boosting，准确率约98%

使用场景： 预估你的论文在未来是否能达到高被引标准

9. All Fields Forecast（全领域预测）

显示内容： 选定出版年份，所有领域的2026年1月预测阈值

排序： 从高到低

🔑 实用查询示例

Q1: 我2021年发表的临床医学论文，有150次引用，现在是高被引吗？

  1. 选择 Field = CLINICAL MEDICINE
  2. 选择 Publication Year = 2021
  3. 查看 "Current Thresholds" 或 "Time Series"
  4. 如果阈值显示 < 150，恭喜！你是高被引

Q2: 未来我的论文还能保持高被引吗？

  1. 查看 "Predicted vs Current"
  2. 如果预测阈值 > 你的引用数，你可能会被淘汰
  3. 继续积累引用！

Q3: 哪个领域最容易成为高被引？

  1. 查看 "Average Threshold by Field"
  2. 阈值最低的领域（如 Mathematics）相对容易
  3. 但这也意味着该领域论文整体引用较少

🚀 如何打开 Dashboard

由于浏览器安全限制，不能直接双击打开 HTML。需要启动本地服务器：

cd "Dashboard_Threshold文件夹路径"
python -m http.server 8080

然后浏览器访问：http://localhost:8080/threshold_dashboard.html
